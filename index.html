# HangOut App 🎉 A smart group hangout planner with real weather integration, AI-scored recommendations, and a lobby voting system. ## Files - `index.html` — full app structure (all screens) - `style.css` — dark neon UI styles - `app.js` — all logic (profiles, sessions, recommendations, weather, maps) ## Features ### ✅ Profile Setup (4 steps) - Name, age, budget slider - Personality tags (multi-select) - Likes & dislikes (cycling: love / nope / neutral) - City + automatic GPS location detection ### ✅ Smart Recommendations Engine Scores 20+ place types based on: - **Age** — older users get calmer venues boosted, younger get energetic ones - **Budget** — hard filters out over-budget places, scores cheaper ones higher - **Personality** — matches traits to venue personalities - **Likes/Dislikes** — boosts loved categories, penalizes disliked ones - **Mood** — your or your group's voted mood affects ranking - **Weather** — live weather from Open-Meteo API adjusts scores: - Rainy → indoor venues rise, outdoor venues drop - Hot → beach/shaded spots rise - Sunny → outdoor venues get bonus - Cold → cozy indoor spots rise ### ✅ Session & Lobby System - Host generates a 6-char code (e.g. `HNG4X2`) - Friends enter code to join the lobby - Live member list with avatars - **Mood vote** — everyone votes their vibe, majority wins and drives recommendations - Simulated member join animations (replace with WebSockets/Firebase for real multiplayer) ### ✅ Google Maps Integration - Map placeholder works out of the box - "Open in Google Maps" button searches your city for the vibe - Clicking any recommendation opens Google Maps search for that place - **To enable live embedded map:** 1.
